# managing-agent-knowledge

A standalone Codex skill for designing, maintaining, auditing, and operationalizing agent-readable knowledge systems.

Install by copying or symlinking this repo into your Codex skills directory, or by installing it through the skill installer workflow used by your Codex setup. The skill entry point is `SKILL.md`; workflows, knowledge pages, and templates are loaded progressively as needed.

The skill is domain-neutral. It distills patterns for source-cited corpora, freshness contracts, closed taxonomies, inclusion-link navigation, retrieval bundles, memory boundaries, and stewardship evals.
